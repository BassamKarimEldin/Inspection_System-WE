Telecom Egypt Inspection System – Web Application

A digital inspection management system designed to replace manual, paper-based workflows used in Telecom Egypt field operations.

Overview

This web application digitizes the inspection process for Telecom Egypt (TE).
It allows inspectors to submit TDM and FTTH inspection forms electronically, while giving administrators full control over user management, inspection data, and operational reporting.
